## 🚀 快速开始

### 环境要求
- Python 3.10+
- CUDA 11.7+
- GPU: RTX 3090/4090 (24GB显存)

### 启动步骤
```bash
# 1. 安装依赖
cd backend
pip install -r requirements.txt

# 2. 启动服务
python main.py

# 3. 访问系统
浏览器打开: http://localhost:8000
```

### 测试功能
```bash
# 提交任务
curl -X POST http://localhost:8000/api/tasks/ \
  -H "Content-Type: application/json" \
  -d '{"prompt": "制作一段关于森林探险的短视频"}'
```
